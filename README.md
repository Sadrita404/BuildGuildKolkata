# Build Guild Kolkata

A beautiful, modern website for Build Guild Kolkata - a free 1-day hardware meetup for teen makers organized by Hack Club Blueprint.

## 🎉 Event Details
- **Date:** Friday, April 18, 2026
- **Venue:** My Cube, Kolkata
- **Location:** Kolkata, India
- **Cost:** FREE

## 🌟 Features

### Hero Section
- Eye-catching hero with event branding
- Quick overview with key event details
- Clear call-to-action buttons

### Navigation
- Smooth scrolling to different sections
- Quick links to Overview, Projects, Sponsors, Organizers, FAQ
- Register button in navigation

### Sections Included

1. **Overview** - Information about what Build Guild is
2. **Featured Projects Showcase** - Rotating carousel of top Hack Club Blueprint projects with:
   - Auto-rotating every 5 seconds
   - Manual navigation controls
   - Direct links to full project pages

3. **Sponsors Grid** - Showcase of supporting organizations:
   - Hack Club
   - GenXYZ
   - EasyEDA
   - JLCPCB
   - OSHWLab

4. **RSVP Section** - Easy registration with perks listing

5. **Organizers** - Meet the team behind Build Guild

6. **FAQ** - Common questions answered

7. **About & Footer** - Additional information and links

## 🛠️ Technical Stack
